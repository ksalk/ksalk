## Hey, I'm Konrad

Software developer based in Warsaw, Poland. I build stuff, break stuff, and occasionally write about it.

I'm primarily focused on the .NET and Azure world, building a wide range of apps, APIs, and CLIs, but I like poking my nose into whatever shiny new tech catches my attention. Some of the technologies I've recently explored include React frameworks, Svelte, and Rust. I'm closely following AI agent tooling and the latest advancements in the field. I've also recently gotten into terminal applications and TUIs.

I write dev blog about .NET and software development, check it out at [ksalk.pl](https://ksalk.pl)
